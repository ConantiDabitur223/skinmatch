## 1. Executive Summary

### 1.1 한 줄 정의

**SkinMatch는 임상 가이드라인 기반의 피부 진단으로 사용자의 질환·트러블 상태를 분류하고, 그에 가장 적합한 활성 성분과 한국 시판 화장품을 추천하는 모바일 앱이다.**

### 1.2 차별점 (Why Now / Why Us)

| 구분 | 기존 서비스(화해, 글로우픽, 룰루랩 LUMINI 등) | SkinMatch |
|---|---|---|
| 진단 깊이 | 사용자 자기보고 또는 단일 AI 스캔 | Baumann 16-type 설문 + AI 셀피 + 트러블 트래킹의 3중 진단 |
| 추천 근거 | 리뷰·랭킹 중심 | PubMed/JAAD 기반 임상 근거 매핑 (성분-기전-효과-증거 등급) |
| 표적 사용자 | 일반 뷰티 소비자 | 피부 질환·만성 트러블(여드름·홍반·아토피·색소침착)을 가진 능동적 관리자 |
| 데이터 수명 | 1회성 진단 | 주기적 재진단 + 사용 제품의 효과 트래킹 |

### 1.3 핵심 가설

1. 한국 소비자의 **64.7%는 맞춤형 화장품을 인지**하고 있고 **85.4%는 추천 의향**이 있다([futuremarketinsights.com](https://www.futuremarketinsights.com/reports/k-beauty-product-market)). 잠재 수요는 충분하다.
2. 그러나 기존 앱은 "리뷰 점수"에 의존하므로, **임상 근거와 피부 진단을 결합한 추천**이라는 빈 영역(white space)이 존재한다.
3. 사용자는 "어떤 제품이 좋은가" 보다 **"내 피부에 왜 이 성분이 필요한가"**를 알 때 전환율과 잔존율이 높아진다.

### 1.4 기대 효과 (12개월)

- MAU 50,000명, 진단 완료율 60%
- 추천 → 제품 클릭(아웃링크/제휴) 전환율 12% 이상
- 평균 진단 재방문 주기 6주
- 어필리에이트/제휴 매출 손익분기 도달

## 2. 시장 분석

### 2.1 한국 화장품 시장 개요

- **2025년 한국 화장품 시장 규모: USD 18.39B**, 2026–2035 CAGR 5.40%로 2035년 USD 31.12B 예상([expertmarketresearch.com](https://www.expertmarketresearch.com/reports/south-korea-cosmetics-market)).
- H&B 스토어가 시장 점유율 54.10%를 차지하나, **온라인 채널이 7.12% CAGR로 가장 빠르게 성장**([mordorintelligence.com](https://www.mordorintelligence.com/industry-reports/south-korea-cosmetics-products-market-industry)).
- 핵심 트렌드: **AI 기반 개인화, 다기능(하이브리드) 제형, 클린 뷰티, 피부장벽 케어**.

### 2.2 맞춤형 화장품 세그먼트

- 2020년부터 식약처가 **맞춤형화장품 판매업** 제도를 정식 시행. 매장에서 **맞춤형화장품 조제관리사** 자격자가 원료를 혼합·소분해 판매할 수 있게 됨([mfds.go.kr](https://www.mfds.go.kr/brd/m_695/list.do)).
- 2025년 조제관리사 자격시험은 5월·9월 연 2회로 확대([cosinkorea.com](https://cosinkorea.com/news/article.html?no=54288)) → 산업적 인프라 성숙기.
- 소비자 인지도 64.7%, 추천 의향 85.4% — **수요 검증 완료된 시장**.

### 2.3 임상적 수요 — 한국인 피부질환 유병률

| 질환 | 유병률/추세 | 출처 |
|---|---|---|
| 아토피피부염 | 전체 인구 2.34%(2011) → 3.33%(2019), **중증 아토피 8년새 2배** | [aard.or.kr](https://aard.or.kr/pdf/10.4168/aard.2024.12.2.58), [mt.co.kr](https://www.mt.co.kr/thebio/2024/07/08/2024070809371815226) |
| 소아 아토피(초등) | 2000년 24.9% → 2010년 35.6% | [amc.seoul.kr](https://www.amc.seoul.kr/asan/healthinfo/disease/diseaseDetail.do?contentId=31519) |
| 여드름 | 청소년 80%+, 성인 여드름 증가 추세(KDA 통계) | KDA 가이드라인 |
| 멜라스마/PIH | 동아시아 여성 다빈도 — Fitzpatrick III–IV 피부에서 PIH 빈도 높음 | [pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC9805721/) |

### 2.4 시장 기회 종합

- **B2C 모바일 앱 + 임상 기반 맞춤 추천**은 룰루랩(LUMINI)이 거울/스캐너 기반 B2B/하드웨어로 점유 중이지만, **순수 모바일 + 셀피 + 설문 기반 임상 매칭 솔루션**은 한국 시장에 비어 있음([lulu-lab.com](https://www.lulu-lab.com/)).
- 화해·글로우픽은 리뷰 큐레이션, 임상 매칭 기능 부재.
- 즉, **"피부과 가기 전 단계에서 자기관리(self-care)를 도와주는 의학적 신뢰감 있는 화장품 코치"**로 포지셔닝 가능.

## 3. 사용자 페르소나 및 문제 정의

### 3.1 1차 페르소나 — "소영, 31세, 직장인 / 성인 여드름 + PIH"

- **상황:** 학창 시절부터 여드름이 잦았고, 회사 스트레스로 턱·이마에 반복 발생. 여드름 후 색소침착이 사라지지 않음.
- **불편한 점:**
  - 인스타·유튜브의 성분 정보가 너무 많아 무엇이 자기에게 맞는지 모름.
  - 피부과는 시간/비용 부담. 처방 화장품 외 일상 케어가 막막.
  - "각질·기름·색소·홍조"가 동시에 있어 기존의 단일 진단(건성/지성)이 무의미.

### 3.2 2차 페르소나 — "민준, 34세, 남성 / 민감성 + 면도 자극"

- **상황:** 면도 후 자극·홍조가 잦고, 피지는 많은데 광대는 건조한 복합성.
- **불편한 점:** 남성 라인은 종류가 적어 시험착오 비용이 큼. 향이 강한 제품은 자극.

### 3.3 3차 페르소나 — "지윤, 47세, 가정주부 / 광노화 + 멜라스마"

- **상황:** 야외활동 후 광대 멜라스마와 잔주름이 신경 쓰임. 자극적인 시술은 부담.
- **불편한 점:** 미백 성분이 많은데 무엇이 더 안전·강력한지, 레티놀·비타민C 병용은 가능한지 헷갈림.

### 3.4 핵심 Pain Points 통합

1. **진단의 빈약함** — "건성/지성"의 단편적 분류만 존재
2. **성분 정보 과잉** — 어느 성분이 내 트러블에 효과적인지 임상적 우선순위 부재
3. **제품-성분 단절** — 성분명을 알아도 그것이 든 한국 시판 제품을 빠르게 찾기 어려움
4. **피드백 루프 부재** — 한 번 사면 끝, 효과 검증 없음

### 3.5 SkinMatch가 해결하는 것 (Job-to-be-done)

> *"내 피부 상태에 맞는 성분이 무엇인지 의학적으로 신뢰할 수 있게 알려주고, 그 성분이 든 한국 시판 제품을 추천하며, 효과를 추적해줘."*

## 5. 서비스 정의 및 차별화 전략

### 5.1 한 문장 정의

> *"피부과 가기 전, 내 피부 상태를 16가지로 분류하고 임상 근거 기반의 성분과 한국 화장품을 매칭해주는 자기관리 코치 앱."*

### 5.2 4대 차별화 축

1. **임상 근거 등급제** — 모든 추천 성분에 SoE(Strength of Evidence) 등급(A/B/C) 표기. JAAD Delphi 컨센서스, Cochrane 리뷰, RCT 인용([jaad.org](https://www.jaad.org/article/S0190-9622(25)00612-7/abstract)).
2. **하이브리드 진단** — Baumann 설문(언어적 자가보고) + AI 셀피(시각적 객관 측정) + 트러블 일지(시계열).
3. **트러블 중심 IA** — UI를 "건성/지성"이 아니라 **"여드름·홍반·색소·민감·광노화·장벽"** 6대 트러블 축으로 재구성.
4. **효과 트래킹 루프** — 추천 후 4·8·12주 시점에 셀피 재촬영 + 만족도 응답으로 추천 모델 보정.

### 5.3 비기능적 차별화

- 광고/리뷰 가중이 아닌 **"성분 적합도 + 안전성"** 점수가 우선
- **이해 가능한 추천(Explainable Recommendation)** — "왜 이 제품인가"를 성분-기전-증거 3계층으로 설명

## 6. 핵심 기능 정의

### 6.1 기능 우선순위 매트릭스 (MoSCoW)

| 우선순위 | 기능 | MVP? |
|---|---|---|
| **Must** | Baumann 설문 진단 (16-type) | ✅ |
| **Must** | 셀피 기반 AI 피부 분석 (모공·홍조·색소·주름·여드름) | ✅ |
| **Must** | 트러블별 성분 추천 + 임상 근거 카드 | ✅ |
| **Must** | 한국 시판 제품 매칭 검색 | ✅ |
| **Must** | 사용 중인 제품 등록 & 성분 충돌 경고 | ✅ |
| **Should** | 4·8·12주 효과 트래킹 (셀피 비교) | Phase 2 |
| **Should** | 데일리 스킨 다이어리 (트러블 위치·증상 기록) | Phase 2 |
| **Could** | 피부과 전문의 Q&A (비대면 상담 제휴) | Phase 3 |
| **Could** | 맞춤형화장품 조제관리사 매장 연계 (오프라인) | Phase 3 |
| **Won't** | 처방의약품 추천 / 진단명 확정 (의료법 위반) | ❌ |

### 6.2 기능 상세

#### F1. 피부 진단 (Baumann + AI Selfie)
- 3–5분 설문 (Baumann 16 type 검증된 설문지 한국어 적응판) ([baumannskintypes.com](https://baumannskintypes.com/), [pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC12163966/))
- 셀피 3장(정면·좌·우) → CNN 모델로 모공·주름·여드름·홍조·색소 5축 점수화
- 결과: **Baumann 4축(O/D, S/R, P/N, W/T) + AI 5축**을 결합한 9-axis Skin Profile

#### F2. 트러블 진단 모듈
- 6대 트러블: **여드름, 홍반/주사, 아토피/장벽, 색소침착(멜라스마/PIH), 광노화, 민감/자극**
- 각 트러블에 대해 임상적으로 검증된 성분 풀(Pool)에서 우선순위 추천

#### F3. 성분 카드
- 성분명(국문/INCI) → 작용기전 → 임상 효과 → SoE 등급 → 권장 농도 → 부작용/금기 → 병용 가능/충돌 성분

#### F4. 제품 매칭
- 제품 DB에서 **(추천 성분 ∩ 사용자 알레르기 제외 ∩ 가격대) → 점수 정렬**
- 유형별 추천: 클렌저 / 토너 / 세럼 / 보습 / 자외선차단

#### F5. 루틴 빌더
- AM/PM 루틴 자동 생성 — 충돌 회피(예: AM 비타민C ↔ PM 레티놀)
- 사용자 편집 가능

#### F6. 효과 트래킹
- D0/D28/D56/D84 셀피 비교, AI 점수 추이 그래프
- 사용자 만족도(NPS) 입력

#### F7. 안전 가드레일
- 임신/수유 → 레티노이드 자동 제외
- 광민감 성분 + 자외선차단 미사용 시 경고
- 알레르기/접촉성 피부염 이력 → 향료/리모넨/리날룰 등 자동 제외

## 7. 피부 진단 엔진: Baumann + AI Selfie 하이브리드

### 7.1 Baumann Skin Type Indicator (BSTI) 적용

- 2004년 Leslie Baumann, MD가 개발한 16-type 분류. **피부과·연구계에서 표준으로 활용**되는 검증 도구([baumannskintypes.com](https://baumannskintypes.com/), [pubmed.ncbi.nlm.nih.gov](https://pubmed.ncbi.nlm.nih.gov/18555952/)).
- 4축 이항 분류:
  - **O / D**: Oily(지성) vs Dry(건성)
  - **S / R**: Sensitive(민감) vs Resistant(저항성)
  - **P / N**: Pigmented(색소) vs Non-pigmented(비색소)
  - **W / T**: Wrinkle(주름) vs Tight(탄탄)
- 16개 조합 (예: OSPW = 지성·민감·색소·주름)
- 검증된 신뢰도: Oily/Dry 분류에서 **정확도 96%, 민감도 95%, 특이도 98%**([pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC12163966/))
- 한국어 번역·문화적 적응 + 사내 IRB 검증으로 시작 가능

### 7.2 AI Selfie 분석

- 셀피 3장 입력 → 전처리(랜드마크 정렬·조명 정규화) → CNN 멀티태스크 모델
- 출력: 모공(0–100), 주름(0–100), 여드름 카운트, 홍조(△E*ab), 색소(불균일도)
- 참고 연구: 피부질환 다중분류 CNN은 95.80–98.21% 정확도 보고. **MobileNet은 모바일 적합 95.7%**([pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC9841357/), [link.springer.com](https://link.springer.com/chapter/10.1007/978-981-19-8032-9_39)).
- 한계: 조명·기기·인종에 따라 일반화 성능 변동 → **외부 검증셋 필수**, 한국인 피부 데이터셋 자체 구축 필요.

### 7.3 결합 로직

- Baumann (구조적·언어적 정보) + AI Selfie (시각적·정량적 정보) → 두 결과의 **불일치(discordance)는 진단 신뢰 신호로 활용**.
- 예: 사용자가 "건성"으로 자가보고했으나 AI가 모공·피지 점수를 높게 측정 → "사용자는 건조감을 느끼지만 객관적으로는 외건성-내지성(脫水性 지성)" 가설 → 보습 + 적당한 BHA.

### 8.1 트러블 ↔ 성분 매핑 (전체 개요)

| **여드름(염증성)** | Salicylic Acid 0.5–2%, Benzoyl Peroxide 2.5–5%, Adapalene 0.1%(OTC) | Niacinamide, Zinc PCA, Azelaic Acid | 코메도제닉 오일(코코넛유 등), 점도 높은 미네랄오일 |
| **여드름(코메도형)** | Salicylic Acid, Adapalene, Retinol | Glycolic Acid, Niacinamide | 두꺼운 폐색 제형 |
| **PIH/색소침착** | Niacinamide 4–5%, Tranexamic Acid 3%, Vitamin C 10–20%, Azelaic Acid 10–20% | Alpha-Arbutin, Kojic Acid, AHA | 자외선차단 미사용 시 모든 미백제 의미 없음 |
| **멜라스마** | Tranexamic Acid 3–5%, Hydroquinone(처방), Azelaic Acid 20% | Niacinamide+TXA+VitC 복합, Cysteamine | 광민감 자극제 단독, 강한 박피 |
| **홍반/주사(rosacea)** | Azelaic Acid 15%, Niacinamide, Centella(Madecassoside) | Allantoin, Bisabolol, Beta-Glucan | 알코올, 향료, 멘톨, 캡사이신, 강한 산 |

| **민감/자극** | Centella, Madecassoside, Panthenol, Beta-Glucan | Allantoin, Bisabolol, Oat extract | 향료, 색소, 알코올, 고농도 산 |
| **광노화/주름** | Retinol 0.1–1.0%, Retinaldehyde, Peptides(매트릭실·아르기렐린), Vitamin C | Bakuchiol(레티놀 대체), AHA, Resveratrol | 자외선차단 미사용 |

### 8.2 핵심 성분별 임상 카드

#### 8.2.1 Niacinamide (나이아신아마이드, Vitamin B3)
- **작용 기전:** 멜라노솜 전이 억제(케라티노사이트 ↔ 멜라노사이트), 피지 분비 조절, 세라마이드 합성 촉진, 항염
- **임상 근거 등급:** A — 톤·홍조·블랙스팟에 대해 피부과 전문의 컨센서스([jaad.org](https://www.jaad.org/article/S0190-9622(25)00612-7/abstract))
- **효과:** 4주 사용 시 색소침착 유의 감소, 멜라노솜 전이 억제 입증([pubmed.ncbi.nlm.nih.gov](https://pubmed.ncbi.nlm.nih.gov/12100180/)). 4% 농도는 멜라스마에서 **약 40% 환자에 우수한 임상 개선**([pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC3142702/)).
- **권장 농도:** 2–10% (일반 5%, 민감성 2–4%부터)
- **부작용/주의:** 저자극이지만 10% 초과 시 일시적 홍조 가능
- **병용:** 비타민C·레티놀과 병용 가능(과거 통설인 충돌설은 근거 부족)

#### 8.2.2 Salicylic Acid (살리실산, BHA)
- **작용 기전:** 지용성 → 모공 내부 침투, 각질 박리, 항염, 피지 용해
- **임상 근거 등급:** A — 여드름·지성 피부에 대한 컨센서스
- **효과:** 5% 나이아신아마이드 + 2% 살리실산 12주 → **콘돔·염증병변 60%, PIH 70% 감소** 사례([cosmoderma.org](https://cosmoderma.org/clinical-approaches-in-vogue-for-combination-therapies-for-acne-and-post-inflammatory-hyperpigmentation-a-comprehensive-review/))
- **권장 농도:** 0.5–2% (일상), 10–30%(피부과 필링)
- **주의:** 임신·수유 중 광범위 사용 회피, 안구 점막 회피

#### 8.2.3 Retinol / Retinoid
- **작용 기전:** 레티노이드 수용체(RAR) 활성화 → 콜라겐 합성 ↑, 표피 turnover ↑, 멜라닌 분포 정상화
- **임상 근거 등급:** A — **인체 임상에서 가장 강력한 근거**의 항노화·여드름·모공 성분([news-medical.net](https://www.news-medical.net/health/Do-Popular-Skincare-Ingredients-Like-Niacinamide-and-Retinol-Actually-Work.aspx))
- **효과:** 잔주름·블랙스팟·여드름·큰 모공·지성에 대해 피부과 전문가 컨센서스([jaad.org](https://www.jaad.org/article/S0190-9622(25)00612-7/abstract))
- **권장 농도:** 0.025–1% retinol / 0.05–0.1% retinaldehyde (점진적 적응 — 주2회 시작)
- **금기:** 임신·수유 중 사용 금지 (자동 제외 로직 필수)
- **주의:** 광불안정 → PM 사용, 자외선차단 필수

#### 8.2.4 Vitamin C (Ascorbic Acid 및 유도체)
- **작용 기전:** 항산화, 티로시나아제 억제 → 멜라닌 합성 ↓, 콜라겐 합성 cofactor
- **임상 근거 등급:** A — 잔주름·다크스팟에 컨센서스
- **권장 농도:** L-AA 10–20%(pH 3.5 이하 안정성 필요), MAP/SAP/AA-2G 등 유도체 5–10%
- **병용 사례:** 10% 나이아신아마이드 + 15% 비타민C → 12주 시 **여드름 80%, 색소침착 85% 개선**([cosmoderma.org](https://cosmoderma.org/clinical-approaches-in-vogue-for-combination-therapies-for-acne-and-post-inflammatory-hyperpigmentation-a-comprehensive-review/))

#### 8.2.5 Tranexamic Acid (트라넥삼산, TXA)
- **작용 기전:** 플라스미노겐→플라스민 전환 억제 → 아라키돈산·프로스타글란딘 ↓ → 멜라노사이트 자극 ↓
- **임상 근거 등급:** A (멜라스마/PIH)
- **효과:** **3% TXA + 1% kojic + 5% 나이아신아마이드 12주 세럼 → PIH 81% 감소, 멜라스마 60% 개선**([pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC12848551/))
- **권장 농도:** 2–5% (국소)
- **장점:** 하이드로퀴논 대체로 안전 프로파일 우수

#### 8.2.6 Azelaic Acid (아젤라산)
- **작용 기전:** 항균(P. acnes), 항염, 티로시나아제 억제
- **임상 근거 등급:** A (여드름·주사·멜라스마 모두)
- **효과:** 15% gel — **12주 후 홍반·염증 병변 유의 감소, 평균 51% 개선 vs 36%(대조군)**([pubmed.ncbi.nlm.nih.gov](https://pubmed.ncbi.nlm.nih.gov/18803456/), [onlinelibrary.wiley.com](https://onlinelibrary.wiley.com/doi/10.1111/jocd.15923))
- **권장 농도:** 10–20%
- **장점:** 임신 중 사용 가능(B등급, 의사 상담 권장)

#### 8.2.7 Ceramide (세라마이드)
- **작용 기전:** 표피 lipid bilayer의 50% 구성 — 장벽 회복, TEWL 감소
- **임상 근거 등급:** A (아토피·건조)
- **효과:** 세라마이드 도미넌트 보습제 + 클렌저 6주 → **아토피 중증도 84% 감소**(vs 50% 대조)([onlinelibrary.wiley.com](https://onlinelibrary.wiley.com/doi/10.1111/dth.14970), [pmc.ncbi.nlm.nih.gov](https://pmc.ncbi.nlm.nih.gov/articles/PMC12604518/))
- **권장 형태:** 세라마이드 NP, AP, EOP의 다중 조합 + 콜레스테롤 + 자유지방산 (3:1:1 황금비)

#### 8.2.8 Hyaluronic Acid (히알루론산)
- **작용 기전:** 자체 무게의 1000배 수분 결합. 분자량별 침투 깊이 다름
- **임상 근거 등급:** A (보습)
- **효과:** 히알루론산 폼 + 세라마이드 크림 비교 RCT — **2주차에 히알루론산 폼이 유의 개선**, 4주에 양쪽 모두 유의([pubmed.ncbi.nlm.nih.gov](https://pubmed.ncbi.nlm.nih.gov/21896129/))
- **활용:** 저/중/고분자 다중 조합으로 표피·진피 양쪽에 작용

#### 8.2.9 Centella Asiatica (병풀, Madecassoside / Asiaticoside)
- **작용 기전:** 콜라겐 합성 촉진, NF-κB 억제(항염), 상처 치유
- **임상 근거 등급:** B — 홍반에 대해 semi-robust([chemistconfessions.com](https://chemistconfessions.com/blogs/how-helpful-is-skincare-for-redness-aka-erythema))
- **권장 농도:** 마데카소사이드 0.1–0.4%
- **K-Beauty 강점:** 한국 시장에 풍부한 라인업(닥터지·바이옴·라보링) → 매칭 ROI 높음

#### 8.2.10 Peptides (펩타이드)
- **작용 기전:** Signal/Carrier/Neurotransmitter peptide 등 종류별로 콜라겐 신호, 구리 운반, 표정주름 이완
- **임상 근거 등급:** B (개별 펩타이드별 편차)
- **대표:** Matrixyl(Palmitoyl Pentapeptide-4), Argireline, Copper Peptide(GHK-Cu)

```
 ├ photosensitizing     (bool)
 ├ comedogenic_score    (0-5)
 ├ allergen_flags[]     (EU 26 향료 알레르겐 등)
 └ references[]         (PMID 링크)

Trouble
 ├ id, name, definition
 ├ first_line_ingredients[]
 ├ second_line_ingredients[]
 ├ avoid_ingredients[]
 └ guideline_refs[]     (KDA, AAD 가이드라인)

User
 ├ baumann_type
 ├ ai_skin_score        {pore, wrinkle, acne, redness, pigment}
 ├ allergies[]
 ├ pregnancy_status
 ├ used_products[]
 └ tracking_history[]
```

### 9.3 추천 점수 함수 (단순화)

```
score(product, user) =
   w1 * IngredientFit(product, user.troubles)
 + w2 * Safety(product, user.allergies, user.pregnancy)
 + w3 * EvidenceGrade(product.key_active)
 - p1 * NegativeFlags(product)        // 향료, 알코올, 코메도제닉 등
 - p2 * ConflictWith(product, user.used_products)
 + w4 * AffordabilityFit(product.price, user.budget)
 + w5 * BrandTrust(product.brand)
```

`w1`–`w5`는 시작 시 휴리스틱(전문가 자문) → 사용자 만족도 데이터로 베이지안 업데이트.

## 12. AI/ML 모델 설계

### 12.1 모델 1 — Skin Score Estimator (셀피 회귀)

- **입력:** 정규화된 얼굴 영역 RGB (224×224)
- **출력:** {pore, wrinkle, acne_count, redness, pigment_unevenness} ∈ [0,100]^5
- **백본:** MobileNetV3-Large (모바일 95.7% 보고치 참고, [link.springer.com](https://link.springer.com/chapter/10.1007/978-981-19-8032-9_39)) → 멀티헤드
- **학습 데이터:** 한국인 피부 셀피 자체 수집 + 합성(StyleGAN 증강) + 공개 데이터셋(SD-198, DermNet) 보조
- **평가:** MAE per axis, ICC vs 피부과 전문의 평정, 외부 검증셋 필수

### 12.2 모델 2 — Skin Type Classifier (Baumann 보정)

- **입력:** 설문 응답 + 모델1 출력
- **출력:** 16 Baumann type 중 다중 라벨 확률
- **방식:** Gradient Boosting (XGBoost/LightGBM) — 설명 가능성 우선

### 12.3 모델 3 — Recommender

- **하이브리드:** Rule-based 안전 필터 → Content-based(성분 적합도) → Collaborative(사용자 만족도) 결합
- **콜드스타트:** 콘텐츠 기반(성분-트러블 매핑)으로 시작, 데이터 누적 후 협업 가중

### 12.4 모델 검증·신뢰

- **예측 신뢰구간 표기** (낮으면 "재촬영 권장")
- **Grad-CAM** 시각화로 사용자에게 "AI가 어디를 봤는지" 보여줌(설명 가능 AI)
- 월 단위 모델 드리프트 모니터링

## 부록 A. 기획자 입력 요청 체크리스트
